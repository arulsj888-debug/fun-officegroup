# Star Trek Team Portfolio - Scroll Animation

An interactive, Star Trek-themed team portfolio website featuring advanced scroll-based animations and character showcases.

## 🚀 Features

- **Scroll-Driven Animations**: Uses CSS `animation-timeline` to create immersive character focus effects as you scroll.
- **Dynamic Character Showcase**: Interactive character profiles that transition between default and "focused" (A23 theme) images.
- **Multi-Page Experience**: Navigate between different team sections (e.g., Team 1 and Team 2).
- **Background Audio**: Integrated Star Trek-inspired audio with autoplay support (requires initial user interaction).
- **Responsive Design**: Optimized for a wide range of devices, from mobile phones to large desktop monitors.
- **Glassmorphism UI**: Sleek, transparent data panels with blur effects and Star Trek aesthetics.

## 🛠️ Usage

Simply open `index.html` in a modern web browser. Use your mouse scroll or touch-drag to navigate through the characters.

## 🌐 Browser Support

This project utilizes modern CSS Scroll-driven Animations. For the best experience, use:
- **Chrome / Edge**: version 115 or higher
- **Safari**: version 17.5+ (with experimental features enabled)
- **Firefox**: Nightly/Experimental support only
