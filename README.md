# Team Portfolio - Scroll Animation

An interactive team portfolio website featuring scroll-based animations and character showcases.

## Features

- Scroll-triggered animations using CSS `animation-timeline`
- Character profiles with image transitions
- Responsive design
- Background audio support
- Smooth character focus effects

## Usage

Simply open `index.html` in a modern browser that supports CSS `animation-timeline` (Chrome 115+, Edge 115+).

## Browser Support

Requires a browser with CSS Scroll-driven Animations support:
- Chrome/Edge 115+
- Safari 17.5+ (with experimental features enabled)
